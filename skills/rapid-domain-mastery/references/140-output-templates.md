# Output Templates

Use the files in `templates/` when a structured deliverable will help:
- `templates/corpus-audit-template.md`
- `templates/field-map-template.md`
- `templates/debate-map-template.md`
- `templates/sprint-plan-template.md`
- `templates/tutor-scorecard.md`

## Default output order
1. Corpus audit
2. Core mental models
3. Major disagreements
4. Open questions
5. Hidden assumptions
6. Prerequisite ladder
7. Diagnostic questions
8. Tutor start or sprint plan

## Formatting rule
Keep it concise but intellectually dense.
Prefer structure that lets the user act immediately.

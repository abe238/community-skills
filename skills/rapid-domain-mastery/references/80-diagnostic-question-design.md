# Diagnostic Question Design

Generate questions that expose whether the user understands mechanisms, constraints, and transfer.

## Required mix
- transfer questions
- compare/contrast questions
- misconception traps
- edge-case or failure-mode questions
- synthesis questions

For each question include:
- why it is diagnostic
- what a shallow answer sounds like
- what a strong answer must include
- what misconception it exposes
- where to review if the user misses it

## Quality test
A memorizer should struggle.
A thinker should have something real to say.

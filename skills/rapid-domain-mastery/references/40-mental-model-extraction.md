# Mental Model Extraction

For each mental model include:

- name
- plain-language explanation
- formal or technical version if relevant
- why experts rely on it
- canonical example
- edge case or failure mode
- most common novice misunderstanding
- source anchors

## Selection rule
Choose models that explain the field's reasoning, not merely its vocabulary.

A good test:
If you removed this model, would expert decisions become much harder to explain?

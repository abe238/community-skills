# Input Contract

Capture the following with minimal friction:

- `subject`
- `goal`
- `current_level`
- `time_budget`
- `corpus_scope`
- `mode`

## Modes
- `field-map`
- `debate-map`
- `tutor`
- `sprint`
- `application`
- `mixed`

Proceed with reasonable defaults if the user did not specify every field.

## Default assumptions
- current_level = beginner-with-some-context
- mode = mixed
- time_budget = unspecified
- corpus_scope = all provided sources

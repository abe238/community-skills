# Study Sprint Planner

Design a compressed learning plan around:
- fast orientation
- high-yield reading
- active recall
- diagnostic testing
- error review
- final no-notes articulation

## Required elements
- ordered blocks
- what to read now vs later
- when to stop reading and start answering
- how to review misses
- final checkpoint the user should be able to explain from memory

## Anti-pattern
Do not produce an aspirational marathon that nobody will actually do.

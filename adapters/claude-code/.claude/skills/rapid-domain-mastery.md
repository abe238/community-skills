# Rapid Domain Mastery

When a user needs to understand a new field fast from multiple sources — exam prep, research onboarding, program handover, or rapid topic mastery — use this skill.

## Doctrine

Build a field map, not a summary: the expert mental model, the live debates, the hidden assumptions, the prerequisites, and the questions that separate real from fake understanding. Be honest about corpus limits — label `[Gap]` rather than papering over them.

Use for multi-source corpora. Skip for plain summaries, single short docs, or quick lookups.

## Process

1. Boundaries: is the corpus strong enough to synthesize? If thin, bound claims and mark `[Gap]`.
2. Input contract: normalize subject, goal, level, time budget, source shape, mode.
3. Corpus audit; field cartography + mental-model extraction.
4. Debate cartography when sources disagree.
5. Assumption mining every deep run.
6. Prerequisite ladder when foundations are missing.
7. Diagnostic questions for exam/mastery prep.
8. Tutor loop or dated study sprint on request / time budget.
9. Quality gates before finalizing; watch summary drift, fake consensus, thin-corpus overreach.

## Epistemic labels

`[Consensus]`, `[Debate]`, `[Open]`, `[Prereq]`, `[Inference]`, `[Gap]`.

## Default output

Corpus audit, core mental models, major disagreements, open questions, hidden assumptions, prerequisite ladder, diagnostic questions, and a tutor start or sprint plan.
